# Changelog

## [1.0.0] - 2026-05-27

### Added
- Full SEO analysis suite (Meta, Headings, Links, Images, Performance, Content)
- Site crawler with configurable depth and concurrency
- Sitemap.xml and Robots.txt generators
- Broken link checker
- Redirect chain detector
- Canonical URL validator
- Mobile-friendly checker
- Competitor analysis with side-by-side comparison
- Keyword rank tracking with history
- FastAPI REST API with full documentation
- Web UI with TailwindCSS + Alpine.js
- CLI with Click + Rich formatting
- Configuration via .env file
